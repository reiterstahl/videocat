# VideoCAT Roadmap

This file tracks public, non-sensitive ideas for future versions of VideoCAT.

## Companion

- Improve Windows packaging and auto-update options.
- Add safer local pairing between the web app and the Windows companion.
- Continue improving monitored disks and manually selected folders.

## Catalog

- Expand duplicate comparison views.
- Add safer bulk actions for tags, deletion queues, and download queues.
- Improve mobile layouts for dense catalog workflows.

## Deployment

- Publish official Docker images for the server and web app.
- Document recommended reverse proxy and HTTPS setups.
- Add a production checklist for secrets, backups, and upgrades.
